# Tool Logos

Add your actual company logos here in SVG or PNG format:

## Required Logos:
- ahrefs.svg (or .png)
- semrush.svg (or .png)
- moz.svg (or .png)

- etc...

## Usage:
Replace the placeholder SVG files with actual company logos downloaded from:
- Company official websites
- https://worldvectorlogo.com
- https://seeklogo.com
- https://brandslogos.com

## Image Format:
- Preferred: SVG (scalable)
- Alternative: PNG (min 256x256px)
- Name format: lowercase, no spaces
